# trabalho-1-redes

Protocolo das mensagens (baseado no Kermit)

- Marcador de início: 8 bits [01111110]
- Tamanho: 6 bits
- Sequência: 4 bits
- Tipo: 4 bits
- Dados: Tamanho em bytes
- Paridade Vertical: 8 bits
